# GoNext

**Дневник туриста** — мобильное приложение для планирования поездок и ведения дневника путешествий. Работает полностью **офлайн**.

## Требования

- Node.js 20.19+
- npm или yarn

## Установка и запуск

```powershell
npm install
npx expo start
```

Откройте приложение в симуляторе/эмуляторе или отсканируйте QR-код в Expo Go.

- **Веб:** `npx expo start --web`
- **Android:** `npx expo start --android`
- **iOS:** `npx expo start --ios`

## Разделы приложения

- **Места** — список мест, добавление, фото, координаты (DD), открытие на карте и в навигаторе.
- **Поездки** — создание поездок, маршрут (порядок мест), отметка посещений, заметки и фото по посещению, текущая поездка.
- **Следующее место** — первое непосещённое место в текущей поездке, карта и навигатор.
- **Настройки** — о приложении.

## Данные и офлайн

- Все данные хранятся **локально**: SQLite (места, поездки, маршруты), фотографии — в `documentDirectory` приложения.
- Работа без интернета: БД и файлы доступны офлайн. Карты и навигатор открываются во внешних приложениях (при наличии сети).
- Ошибки доступа к БД и файлам обрабатываются (например, при сохранении фото при ошибке возвращается `null`).

## Технологии

- Expo SDK 55, Expo Router, React Native Paper, TypeScript
- SQLite (expo-sqlite), expo-image-picker, expo-file-system
